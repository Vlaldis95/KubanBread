Инструкция по установке Django на компьютер.
Все операции для OС Windows.
Для других ОС ищите информацию по установке на сайте https://metanit.com/python/django/1.2.php

1) Открыть данный проект на компьютере с помощью VsCode. 
Перейти с помощью терминала в директорию Kubansite/

2) Установить python в VsCode. Для этого в терминале ввести команду: 

pip install python

3) Установить виртуальное окружение.Для этого в терминале ввести команду::

python -m venv venv
В директории KubanBread/ должна появиться скрытая папка под названием venv

4) Далее необходимо активировать виртуальное окружение командой в терминале:

source venv/Scripts/activate

После 4 этапа в терминале в начале командной строки должно появиться обозначение "(venv)" 
Это значит, что виртуальное окружение установлено успешно

5) И наконец, установка самого Django. Снова ввести в терминале:

pip install django

После 4го этапа необходимо проверить установку всех утилит.
Для этого в терминале ввести команду:

pip list

После этой команды в консоли выйдет список установленных программ. 
Должен быть установлен python версии 3.9 и Django версии 4.1. Если это так, то все нужные утилиты установлены

!!!!! Для запуска локального сервера и просмотра работоспособности сайта необходимо перейти в директорию kubansite/kubansite 
( в ту директорию, где лежить файл manage.py ) и выполнить команду в терминале:

python manage.py runserver

После этого смотреть терминал, если нет ошибок, то можно смотреть работу сайта по адресу http://127.0.0.1:8000/
